# Text Matching Based on LCQMC: A Large-scale Chinese Question Matching Corpus


### 模型
* ABCNN
* BIMPM
* DSSM-BOW
* DSSM-CNN
* DSSM-RNN
* DSSM-Embedding
* ESIM
* BM25
* Edit-Distance
* Jaccard

### 数据形式：
